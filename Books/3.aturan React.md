# ada Beberapa Aturan di Dalam React yang harus kamu patuhi

## Format JSX 

| **Aturan JSX**                              | **Penjelasan / Perbedaan dari HTML**                      | **Contoh JSX**                                             | **Catatan Tambahan**                                      |
| ------------------------------------------- | --------------------------------------------------------- | ---------------------------------------------------------- | --------------------------------------------------------- |
| **Gunakan `className` bukan `class`**       | Karena `class` adalah keyword di JavaScript.              | `<div className="container">Hello</div>`                   | React akan ubah `className` jadi `class` di output HTML.  |
| **Gunakan `htmlFor` bukan `for`**           | `for` juga keyword di JS, jadi React ubah jadi `htmlFor`. | `<label htmlFor="email">Email:</label>`                    | Sama seperti di `<label for="email">`.                    |
| **Satu root element**                       | JSX harus dibungkus dalam satu elemen induk.              | `<div><h1>Hi</h1><p>Hello</p></div>`                       | Bisa juga pakai `<></>` (Fragment).                       |
| **Gunakan `{}` untuk ekspresi JS**          | Di dalam JSX, ekspresi JS ditulis dengan `{}`.            | `<h1>{user.name}</h1>`                                     | Bisa juga untuk perhitungan: `{2 + 2}`.                   |
| **Gunakan `camelCase` untuk atribut**       | Beberapa atribut berubah gaya penulisannya.               | `<button onClick={handleClick}>Click</button>`             | Contoh lain: `tabIndex`, `readOnly`, `maxLength`.         |
| **Harus ditutup**                           | Semua tag harus ditutup, termasuk tag tunggal.            | `<img src="logo.png" />`                                   | Kalau di HTML boleh `<img>`, di JSX wajib `<img />`.      |
| **Gunakan `style` sebagai objek JS**        | Gaya inline ditulis sebagai objek.                        | `<div style={{ color: 'red', fontSize: '20px' }}>Hi</div>` | Nilai numerik otomatis jadi `px`.                         |
| **Komentar di JSX pakai `{/* ... */}`**     | Karena JSX dikompilasi ke JS, bukan HTML.                 | `{/* Ini komentar */}`                                     | Kalau di dalam `{}` biasa, React bakal error.             |
| **Nama komponen harus huruf besar di awal** | React bedain komponen dan elemen HTML dari huruf pertama. | `<MyButton />` vs `<button>`                               | Huruf besar = komponen kustom. Huruf kecil = elemen HTML. |
