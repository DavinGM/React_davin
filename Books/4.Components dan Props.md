
### üîπ **1. Components**

**Components** adalah **bagian terkecil dan paling penting dari React**.
React membangun tampilan (*UI*) menggunakan potongan-potongan kecil yang disebut **komponen**, dan setiap komponen bisa digunakan kembali di mana saja dalam aplikasi.

Komponen dapat dianggap seperti **fungsi JavaScript** yang:

* menerima input (bisa lewat *props*),
* dan mengembalikan elemen React (biasanya dalam bentuk JSX).

Ada dua jenis utama komponen:

1. **Functional Component** ‚Üí ditulis menggunakan fungsi (paling umum sekarang).
2. **Class Component** ‚Üí ditulis menggunakan class (jarang digunakan di versi modern).
3. **Const Arrow** ‚Üí di tulis dengan menggunakan Variable 

 ***Contoh Class Compoents***
 ```jsx
 import React, { useState } from "react";
 ```

### üîπ **2. Props (Properties)**

**Props** adalah singkatan dari **Properties**, yaitu **data yang dikirim ke komponen**.
Props digunakan agar satu komponen bisa menjadi dinamis ‚Äî tidak hanya menampilkan data statis.
```jsx
import React, { useState } from "react";





// const Arrow

const Button = (props) => {
  return(
    <button
        className={`h-10 px-6 font-semibold rounded-md ${props.variant} text-white`}
        type="submit"
      >
        {props.text}
      </button>
  )
}


function App() {

  return (
    <div className="flex justify-center items-center min-h-screen bg-blue-600">
      <div className="flex gap-3">
        <Button variant="bg-black" text="login" />
        <Button variant="bg-red-700" text="logout" />
        <Button variant="bg-red-700"text="nairha" />

      </div>

    </div>
  );
}

export default App;

```

Props dikirim seperti *atribut HTML*, tapi dalam React ditulis menggunakan format JSX:

```jsx
<Greeting name="Nairha" />
```

Dan di dalam komponen, props tersebut diterima sebagai parameter:

```jsx
function Greeting(props) {
  return <h1>Hello, {props.name}!</h1>;
}
```

Maka output-nya adalah:

```
Hello, Nairha!
```

Props **tidak dapat diubah dari dalam komponen** (bersifat *read-only*).
Kalau kamu ingin data berubah, biasanya menggunakan konsep lain seperti *state* (nanti kita bahas di sesi berikutnya ).

### Memberikan Nilai Default pada Props

Apabila Kita Membuat sebuah Compoents yang apabila lupa Memberikan sebuah Property atau Nilai di dalam nya kita bisa melakukan antisipasi agar tidak terjadi bug jikalau Compoents Tersebut di render tanpa punya Propery sama sekali . dari pada membiarkan nya terender lebih baik kita beri Nilai Defualt agar bisa terlihat 

tambahkan ini di bawah Components kamu 
```jsx
const { children = "{...}", variant = "bg-black" } = props
```
> ini artinya jika kamu punya Property Children dan Variant di dalam Components kamu maka isi di dalma nya akan di ubah menjadi nilai "{...}" dan "bg-black" selagi belum ada nilai lain yang akan menumpuk 

Sisa nya kamu tinggal ubah nilai yang punya key `props.`di project kamu contoh
```jsx
className={`h-10 px-6 font-semibold rounded-md ${props.variant} text-white`}
```
ubah menjadi
```jsx
className={`h-10 px-6 font-semibold rounded-md ${variant} text-white`}
```
